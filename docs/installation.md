# Установка

1. Клонируйте репозиторий:
   ```sh
   git clone https://github.com/yourname/voice-widget.git
   cd voice-widget/app
   ```
2. Установите зависимости:
   ```sh
   npm install
   ```
3. Скопируйте `.env.example` в `.env` и заполните параметры:
   ```sh
   copy .env.example .env
   ```
4. Запустите сервер:
   ```sh
   npm start
   ```

Для разработки используйте:
```sh
npm run dev
```

Для тестирования:
```sh
npm test
```
